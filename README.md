# Vite Template

Template Vite & Vue con :

1. Bootstrap
2. Sass
3. Axios
4. VueRouter

# Passaggi installazione

1. Clonate la repo da github col comando

```
git clone https://github.com/SimoneCalzari/comus-frontend.git
```

2. Installate le dipendenze per lo sviluppo

```
npm install
```

3. Lanciate il server di sviluppo

```
npm run dev
```
